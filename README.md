# Sai Sagar Chilakapati — Portfolio

Personal portfolio site built with plain HTML, CSS, and JavaScript, styled with the **Salesforce Lightning Design System (SLDS)** and a custom Cosmos-inspired theme.

## Live Site

**<https://chilsai.github.io/MyPortfolio/>**

## Tech Stack

| Layer | Details |
|-------|---------|
| Markup | Semantic HTML5 |
| Styling | SLDS 2.29 + custom Cosmos-theme CSS variables |
| Interactivity | Vanilla JS (Intersection Observer, smooth scroll) |
| Hosting | GitHub Pages (static) |

## Project Structure

```
MyPortfolio/
├── index.html                 # Main single-page portfolio
├── blog/
│   ├── architect-to-agentblazer.html
├── .nojekyll                  # Bypass Jekyll on GitHub Pages
├── .gitignore
└── README.md
```

## Local Development

Open `index.html` in any browser — no build step required.
